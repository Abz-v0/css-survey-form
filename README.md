# Survey Form Project

My second CSS learning project: a responsive feedback survey form.

## What I built
- HTML form with text, email, number, select dropdown, checkboxes, textarea
- Styled card layout with Flexbox
- Focus and hover effects
- Mobile-friendly design (media queries)

## Technologies
- HTML5
- CSS3 (Flexbox, box-shadow, transitions)
